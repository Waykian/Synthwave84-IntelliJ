### ⚡ Welcome to the Grid

Immerse yourself in the 1984 that never was. Analog synthesis, retina-burning neon colors, and that late-night glow that makes you feel like a hacker breaking into the Tyrell Corporation mainframe.

**Why this fork?** 
The original JetBrains port was legendary, but it was abandoned. It eventually broke and became incompatible with **IntelliJ IDEA 2026.1+**. 
We didn't just apply a band-aid. We ripped out the old legacy engine, rebuilt the entire architecture from scratch using the modern **IntelliJ Platform Gradle Plugin 2.x**, compiled it against **Java 21**, and pumped the neon colors to the absolute maximum. 

This is the **Neon Overdrive Edition**. Fast, stable, and immortal.

### ✨ Features

- 🚀 **Future-Proof Architecture:** Rebuilt from the ground up for IntelliJ 2026.1 and beyond. Completely air-gapped from the marketplace to prevent unwanted overwrites.
- 💖 **Pure Neon Injection:** Stripped away washed-out pastels. Upgraded with true glowing hex codes (Miami Pink, Pure Cyan, and Toxic Green).
- 🧬 **Matrix-Ready:** Fully optimized for Semantic Highlighting and custom cursive fonts.
- 🕶️ **New UI Compatible:** Seamless integration with JetBrains' modern interface and Islands.
- 🥷 **Ghost Hints:** Removed the ugly gray background from inline parameter hints, leaving only floating cyan text.

### 💾 Installation

Since this is a custom, heavily modified build, you won't find it on the standard marketplace. You have to sideload it.

1. Go to the [Releases](https://github.com/Waykian/Synthwave84-IntelliJ/releases) page of this repository.
2. Download the latest `synthwave84-jetbrains-x.x.x.zip` file. **Do NOT extract it.**
3. Open your JetBrains IDE (IntelliJ, WebStorm, PyCharm, etc.).
4. Navigate to **Settings** (or Preferences) → **Plugins** → Click the Gear Icon ⚙️.
5. Select **`Install Plugin from Disk...`** and choose the `.zip` file you downloaded.
6. Click **Restart IDE**.
7. Go to **Settings** → **Appearance & Behavior** → **Appearance** → Select **SynthWave '84 (Neon Overdrive)** from the Theme dropdown.

### 🔮 The Setup Ritual

To get the exact look and feel of a true retro-hacker terminal, the theme alone isn't enough. You need to configure your IDE's engine. Apply these settings:

#### 1. The Hacker Font (Cursive & Ligatures)
This theme is specifically designed to highlight keywords and comments in *cursive italics*.
* Download and install a developer font that supports cursive italics, like [**Victor Mono**](https://rubjo.github.io/victor-mono/) (Highly Recommended) or **Fira Code**.
* In your IDE, go to **Settings → Editor → Font**.
* Select your new font and **check the box for "Enable ligatures"** (this turns `!=` or `->` into beautiful single symbols).

#### 2. The Matrix Effect (Semantic Highlighting)
Make every single local variable glow with a slightly different neon hue.
* Go to **Settings → Editor → Color Scheme → Language Defaults**.
* Scroll down to **Semantic highlighting** and check the box to enable it.

#### 3. Cyberpunk Background (Optional)
* Press `Shift` twice to open *Search Everywhere*.
* Type `Set Background Image`.
* Choose a retrowave grid or cyberpunk skyline image, set the opacity to **5-10%**, and let it bleed through your code.
