### ⚡ Benvenuto nella Griglia

Immergiti nel 1984 che non è mai esistito.  
Sintesi analogica, colori al neon che bruciano lo schermo, e quel glow che ti fa sentire dentro un film cyberpunk.

Questo fork è nato perché la versione originale non era più compatibile con **IntelliJ 2026.1**.  
Ora è aggiornata, stabile e pronta a farti codare come un hacker di Night City.

### ✨ Caratteristiche

- **Colori fedeli** all’originale SynthWave '84 di Robb Owen
- Neon intensi su keyword, stringhe, funzioni e commenti
- Palette perfetta per coding notturno
- Compatibile con **New UI** e **Islands**
- Leggero e velocissimo
- Totalmente open source (GPL-3.0)

### 🚀 Installazione

1. Scarica l’ultima release da [Releases](https://github.com/Waykian/synthwave84-jetbrains/releases)
2. In IntelliJ IDEA vai su **Settings → Plugins → ⚙️ → Install plugin from disk...**
3. Seleziona il file `.zip`
4. Riavvia l’IDE
5. Vai in **Settings → Editor → Color Scheme** e seleziona **SynthWave '84**

### 🎮 Extra consigliati

- **Gif Background** → per mettere una griglia synthwave o pioggia al neon come sfondo
- Font consigliato: **JetBrains Mono** o **Fira Code** con ligature attivate
- Per un glow ancora più estremo → duplica il tema e aumenta la luminosità dei colori
